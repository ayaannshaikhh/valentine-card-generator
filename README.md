# Minecraft Valentine's Card Generator

A simple Minecraft-themed Valentine's Day card generator built with HTML, CSS, and JavaScript.

This project lets users write a custom Valentine’s message and download it as a shareable image styled with a pixel-art Minecraft aesthetic.

The app runs entirely in the browser — no backend or database required.

---

## Features

- Minecraft-style UI using the Minecraftia font
- Customizable Valentine’s message
- Multi-line letter support
- Pixel-art hearts and decorations
- Soft sky background with floating clouds
- Optional watermark behind the letter
- Export card as a PNG image
- Lightweight static website

---

## Usage

Open `index.html` in a browser, write your message, and click **SAVE LETTER** to download the card.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- html2canvas
